There is ``at`` notation in JABS. Ex.: quicklisp@repository@plugin@jabs

root namespace: jabs
tools:   tools@jabs
core:    core@jabs
plugins: plugin@jabs

Plygin types:
repository
test
packaging
...
etc

Package namespaces used:


base packages:
jabs
logger@jabs - logger library
regexp@jabs - regexp library
tools@jabs - jabs tools
core@jabs - core package
